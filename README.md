
TilEm-Qml
=========


Cloning
-------

    git clone https://github.com/labsin/TilEm-Qml.git --recursive

Build Dependencies
------------

Are listed as packages on Ubuntu.

*   libglib2.0-dev
*   libticonv-dev
*   libticalcs-dev
*   libgdk-pixbuf2.0-dev

Build instructions
------------------

    mkdir build
    cd build
    cmake ../build
    make

To run inside build dir:

    make run
